metasploit官方安装教程

· 2020-03-18 · # metasploit
  1. 推荐使用官方安装脚本进行安装
  2. 首先执行如下命令，执行如下命令后会下载安装脚本更显安装源进行自动安装
    curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall
    chmod 755 msfinstall && ./msfinstall
  1. sudo apt install postgresql #安装postgresql
  1. sudo service postgresql status #查看数据库服务是否开启
  1. msfconsole #终端输入命令开启msf

  1. msfdb init #初始化数据库

  2. 搜索任意无警告，安装完成
